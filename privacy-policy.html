Privacy Policy

Effective Date: February 4, 2026

Kafić.hr ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how our Chrome Extension and Web App handles your information. By using Kafić.hr, you agree to the practices described in this policy.

1. Data Collection and Storage

We do not collect, transmit, or store your personal data on any external servers.

Kafić.hr operates on a "Local-First" and "Peer-to-Peer" (P2P) architecture. All data generated by the application (including menus, orders, session logs, and settings) is stored exclusively on your local device using standard browser technologies (LocalStorage and IndexedDB).

2. Data Synchronization

To enable real-time collaboration between devices (e.g., between a waiter and a bartender), Kafić.hr uses WebRTC technology via the Trystero library. This allows devices to connect directly to one another without passing data through a central database.

  • Signaling: A lightweight signaling server is used temporarily only to establish the initial connection ("handshake") between devices using a secure room code. No application data (orders, names, menus) persists on this signaling server.
  • Encryption: Data transmitted between peers is encrypted using standard WebRTC security protocols (DTLS/SRTP).

3. Permissions

The Kafić.hr Chrome Extension requires specific permissions to function:

  • sidePanel: Used to display the application interface in the browser's side panel for persistent access.
  • storage: Used to save your settings, menu configuration, and active session state locally on your device.
  • camera (Optional): Requested only if you choose to scan a QR code to join a session. Video data is processed locally in memory and is never recorded or sent anywhere.

4. Third-Party Services

This application is self-contained. It does not integrate with third-party analytics, tracking services, or advertising networks.

5. Your Rights

Since all data is stored locally on your device, you have full control over it. You can delete all application data at any time by clearing your browser's cache/storage for the extension or by using the "Factory Reset" button within the application settings.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Effective Date".
